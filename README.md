# Risa AI Asisten

Asisten AI pribadi Rivaldo — chatbot cerdas berbasis Puter.js + Google Gemini.

## Fitur

- **Chat interaktif** dengan Risa, asisten AI pribadi
- **Dark mode** dengan toggle yang persist di localStorage
- **Responsive** — desktop split-screen & mobile stack
- **Animasi modern** — mesh gradient, spotlight button, skeleton shimmer
- **Dua tab** — Chat dan Riwayat percakapan

## Teknologi

- HTML + CSS (CSS variables, animasi spring)
- JavaScript (Puter.js SDK)
- Google Gemini API (via backend Express opsional)
